# Phase1 웹앱

## 📋 프로젝트 개요

퍼스널컬러 진단 및 화장품 매칭 웹앱

## 🚀 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 http://localhost:3000 접속

## 📁 프로젝트 구조

자세한 구조는 [PROJECT_STRUCTURE.md](./PROJECT_STRUCTURE.md) 참고

### 주요 기능 구조
- ✅ 인증 시스템 (소셜 로그인, OS별 분기)
- ✅ 홈 페이지 (동적 콘텐츠, 백오피스 연동)
- ✅ 퍼컬매칭 (3단계 선택 프로세스)
- ✅ 스킨매칭 (구조 준비 완료)
- ✅ 화장품매칭 (구조 준비 완료)
- ✅ GNB 네비게이션

### 기술 스택
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: CSS Modules
- **State Management**: React Hooks
